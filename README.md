# 🔬 Анализатор руд

Автоматизированный веб-инструмент для сегментации, классификации и количественного анализа микроструктур сульфидных руд. Приложение генерирует детальные PDF-отчеты с графиками, визуализацией масок и оценкой обогатимости по каждому загруженному изображению.

## ✨ Возможности

- **Zero-shot классификация:** Использование SAM-vit-b + CLIP ViT-B/32 для распознавания минералов без предварительного обучения.
- **Анализ замещения:** Автоматическое определение типа срастаний сульфидов (обычные, тонкие, промежуточные) для оценки труднообогатимости.
- **Детальная статистика:** Расчет площадей, процентных долей фаз и групп (сульфиды, оксиды, тальк, матрица).
- **Генерация отчетов:** Создание профессиональных PDF-документов с таблицами, графиками распределения фаз, легендами и визуализацией патчей срастаний.
- **Предобработка:** Встроенные инструменты CLAHE, шумоподавления и нормализации освещения.

## 📋 Требования

- Python 3.10+
- CUDA-capable GPU (рекомендуется 6GB+ VRAM) или CPU (медленнее)
- Оперативная память: 8GB+

## ️ Установка и запуск

> **⚠️ ВАЖНО:** Настоятельно рекомендуется использовать виртуальное окружение (`venv`) для изоляции зависимостей проекта и предотвращения конфликтов версий библиотек (особенно NumPy/Pandas/Torch).

## 1. Клонирование репозитория
``` bash 
git clone https://github.com/biritva/ora-segmentation.git
cd ora-segmentation 
```

## 2. Создание виртуального окружения
### Windows
``` bash 
python -m venv .venv
.venv\Scripts\activate
```
### Linux / macOS
``` bash 
python3 -m venv .venv
source .venv/bin/activate
```
## 3. Установка зависимостей
``` bash 
pip install -r requirements.txt
```
## 4. Загрузка весов моделей
При первом запуске приложение автоматически скачает веса SAM (~375 МБ). Если у вас ограниченный интернет, скачайте вручную и положите в корневую папку проекта:
``` bash 
wget https://dl.fbaipublicfiles.com/segment_anything/sam_vit_b_01ec64.pth
```
