# 🔬 Анализатор руд

Автоматизированный веб-инструмент для сегментации, классификации и количественного анализа микроструктур сульфидных руд. Приложение генерирует детальные PDF-отчеты с графиками, визуализацией масок и оценкой обогатимости по каждому загруженному изображению.
# Обращаем внимание для развертывания приложения mosel перейдите в папку ora-segmentation/mosel и следуйте инструкциям.
## Далее описание решения ora-segemntation
## ✨ Возможности

- **Zero-shot классификация:** Использование SAM-vit-b + CLIP ViT-B/32 для распознавания минералов без предварительного обучения.
- **Анализ замещения:** Автоматическое определение типа срастаний сульфидов (обычные, тонкие, промежуточные) для оценки труднообогатимости.
- **Детальная статистика:** Расчет площадей, процентных долей фаз и групп (сульфиды, оксиды, тальк, матрица).
- **Генерация отчетов:** Создание профессиональных PDF-документов с таблицами, графиками распределения фаз, легендами и визуализацией патчей срастаний.
- **Предобработка:** Встроенные инструменты CLAHE, шумоподавления и нормализации освещения.

## 📋 Требования

- Python 3.10+
- CUDA-capable GPU (рекомендуется 6GB+ VRAM) или CPU (медленнее)
- Оперативная память: 8GB+

## 📊 Структура PDF-отчета

Каждое обработанное изображение получает индивидуальный отчет, содержащий:

- **Статистика по классам:** Таблица с количеством объектов, площадью (px²/мкм²) и долей для каждого минерала.
- **Статистика по группам:** Агрегированные данные по сульфидам, оксидам, тальку, матрице и структурным элементам.
- **Графики распределения:** Горизонтальная столбчатая диаграмма долей фаз и круговая диаграмма распределения площадей.
- **Визуализация с масками:** Оригинальное фото с наложенными цветными масками и легендой, отображающей только найденные классы с их количеством.
- **Анализ замещения:** Таблица типов срастаний и визуализация патчей сульфидов с цветовой индикацией степени замещения нерудной фазой.
- **Итоговая сводка:** Индекс труднообогатимости, тип руды и рекомендации по схеме обогащения.

  ### Пример сформированного отчета
  "/supp_mat/ora-segmentation_report.pdf"

## 🛠️ Настройки предобработки

В боковой панели приложения доступны параметры для адаптации под разные типы микрофотографий:

- **CLAHE:** Улучшение локального контраста.
- **Шумоподавление:** Non-local means фильтрация.
- **Масштабирование:** Ограничение максимального разрешения для панорамных снимков.
- **Мин. площадь сегмента:** Фильтрация мелкого шума при сегментации.

## ️ Архитектура

Проект построен на стеке:

- **Frontend:** Streamlit (интерактивный веб-интерфейс)
- **Segmentation:** Segment Anything Model (SAM-vit-b)
- **Classification:** CLIP (ViT-B/32)
- **Image Processing:** OpenCV, scikit-image, Pillow
- **Reporting:** ReportLab (генерация PDF)
